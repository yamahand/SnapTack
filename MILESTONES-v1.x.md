# SnapTack マイルストーン (v1.x)

MILESTONES.md (M1〜M6) の続き。詳細仕様は SPEC-v1.x.md を参照。
易→難の順に並べており、M7+M8 完了時点で v1.1、M9 で v1.2、M10 で v1.3 をリリースできる。

## M7: PNG 保存 (→ v1.1 前半)

**ゴール: 付箋を PNG ファイルとして保存できる**

- [x] コンテキストメニューに「PNGで保存...」追加 (`Ctrl+S`)
- [x] SaveFileDialog(デフォルト名 `SnapTack_yyyyMMdd_HHmmss.png`)
- [x] PngBitmapEncoder で物理ピクセル等倍保存
- [x] 前回保存フォルダを設定ファイル(`LastSaveDirectory`)に記憶、初回は「ピクチャ」
- [x] 保存失敗時のエラー処理(付箋は維持)

**確認**: 保存した PNG のピクセルサイズが選択時のサイズ表示と一致する

## M8: 不透明度変更 (→ v1.1 完成)

**ゴール: 付箋の透け具合をホイールで変えられる**

- [x] 付箋上のマウスホイールで不透明度 ±10%(範囲 20〜100%。10% の倍数以外からは最初の 1 ステップを ±5% として倍数へスナップ)
- [x] コンテキストメニューにプリセット 100/75/50/25%(現在値にチェック)
- [x] `Window.Opacity` に適用。コピー / PNG 保存は影響を受けないことを確認
- [x] 新規付箋は常に 100%

**確認**: 20% まで下げても操作(ドラッグ・右クリック・ホイールで戻す)ができる

## M9: サイコロ化 (→ v1.2)

**ゴール: 付箋を小さなタイルに畳んで場所を節約できる**

- [x] ダブルクリックで 48×48 DIP のサイコロ ⇔ 元サイズをトグル
- [x] サイコロはサムネイル表示(アスペクト比維持・中央クロップ)+枠線 1px
- [x] 位置は左上基準で維持(復元時に画面外へはみ出す場合はモニタ内へクランプ)
- [x] サイコロ状態でも移動 / 中クリック閉じる / メニュー / Ctrl+C / Ctrl+S が機能
- [x] コンテキストメニューに「サイコロ化 / 元に戻す」(状態で表記切替)
- [x] ダブルクリック判定とドラッグ移動が干渉しないこと
- [x] 不透明度の維持

**確認**: サイコロ化→移動→復元で、期待した位置・サイズに戻る

## M10: マルチディスプレイ (→ v1.3)

**ゴール: 全モニタでキャプチャ・付箋表示ができる**

- [x] 着手前に現行 `StartCapture` フローをリファクタリング(スコープはキャプチャ起動フローに限定)
- [x] `IScreenCapturer` を「モニタ列挙+指定モニタキャプチャ」に拡張
- [x] モニタごとにフリーズキャプチャ+個別オーバーレイを生成
- [x] 選択は 1 モニタ内で完結(他モニタへはみ出したらクリップ)
- [x] どこかで確定 / キャンセルしたら全オーバーレイを閉じる
- [x] 付箋はキャプチャ元モニタの元位置・元 DPI 基準で生成
- [ ] 混在 DPI 環境(150%+100% 等)での位置・サイズ検証
- [ ] 付箋を別モニタへドラッグ移動できることの確認
- [x] DPI 挙動設定 `KeepPhysicalPixelSize` の追加(既定: WPF 既定動作)+ 設定画面にトグル追加
- [x] 等倍維持モード時の `WM_DPICHANGED` 独自処理

**確認**: セカンダリモニタでキャプチャ→付箋が元位置に重なる。両モニタの DPI が異なっていてもズレない。DPI 挙動設定の両モードで、別モニタへ移動した際の挙動が仕様どおり

## 実装順の補足

- M7 と M8 は独立しており並行可能だが、リリースは v1.1 としてまとめる
- M10 はオーバーレイ生成まわりの構造変更を含むため最後に回している。着手前に現行の `StartCapture` フローのリファクタリングを実施する(ただしスコープはキャプチャ起動フローに限定)
