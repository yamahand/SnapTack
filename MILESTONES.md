# SnapTack マイルストーン

各マイルストーンは「単体で動作確認できる状態」をゴールとする。上から順に実装する。

## M1: プロジェクト基盤 + トレイ常駐

**ゴール: 起動するとトレイに常駐し、メニューから終了できる**

- [ ] ソリューション / プロジェクト作成 (.NET 10, WPF, `net10.0-windows`)
- [ ] app.manifest で Per-Monitor V2 DPI を宣言
- [ ] Mutex による二重起動防止
- [ ] `ShutdownMode=OnExplicitShutdown`、起動時にウィンドウを出さない
- [ ] トレイアイコン表示 + メニュー(キャプチャ / 設定(仮・無効) / 終了)
- [ ] 終了処理 (トレイアイコン・リソースの破棄)

**確認**: 起動→トレイに出る→2つ目を起動しても増えない→終了できる

## M2: グローバルホットキー

**ゴール: Ctrl+Shift+Z で任意の処理(仮メッセージ)が発火する**

- [ ] `RegisterHotKey` の P/Invoke ラッパー (メッセージ専用ウィンドウで WM_HOTKEY 受信)
- [ ] Ctrl+Shift+Z 登録、`MOD_NOREPEAT` 指定
- [ ] 登録失敗時の警告表示(アプリは継続)
- [ ] Dispose で確実に登録解除

**確認**: 他アプリにフォーカスがあってもホットキーが効く

## M3: 画面キャプチャ + 範囲選択オーバーレイ

**ゴール: ホットキー→画面フリーズ→範囲選択→選択範囲の画像が取得できる**

- [ ] キャプチャ抽象化 (`IScreenCapturer`) + `CopyFromScreen` 実装(プライマリモニタ、物理ピクセル)
- [ ] 全画面オーバーレイ(フリーズ画像表示、枠なし、Topmost)
- [ ] 薄暗レイヤ + 選択範囲のくり抜き表示
- [ ] 選択枠(1px)とサイズラベル(物理 px 表記)
- [ ] Esc / 右クリックでキャンセル、極小ドラッグはキャンセル扱い
- [ ] DIP→物理ピクセル変換して `CroppedBitmap` で切り出し
- [ ] オーバーレイ表示中の再入防止

**確認**: 選択した範囲が正しく切り出される(125%/150% スケーリングでも)

## M4: 付箋 (スクラップ) ウィンドウ

**ゴール: v1.0 のコア体験が完成する**

- [ ] 枠なし・枠線1px・影なし・Topmost・タスクバー非表示のウィンドウ
- [ ] キャプチャ元と同じ位置に等倍で表示(物理 px→DIP 変換)
- [ ] 左ドラッグで移動
- [ ] Ctrl+C でクリップボードへコピー(失敗時の例外処理)
- [ ] 中クリックで閉じる
- [ ] 右クリックメニュー(コピー / 閉じる)
- [ ] 複数枚の同時存在、全部閉じてもアプリ継続

**確認**: キャプチャ→付箋がその場に残る→動かせる→コピーできる→閉じられる

## M5: 設定画面 + 設定永続化

**ゴール: ホットキーを変更して保存できる**

- [ ] Fluent テーマ適用 (`ThemeMode="System"`)
- [ ] ホットキー変更 UI (キー入力キャプチャ方式)
- [ ] JSON 設定ファイル(exe 同階層、書き込み不可なら `%APPDATA%\SnapTack` へフォールバック)
- [ ] 変更したホットキーの即時再登録

**確認**: ホットキー変更→再起動しても維持される
